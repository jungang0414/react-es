# 請使用電腦版瀏覽器開啟

## 下載&使用 (本地)

克隆
```
git clone https://github.com/jungang0414/react-es.git
```

下載依賴檔案

```
npm install
```

運行
```
npm run dev
```

## 使用技術

vite&React 創建React

- 路由 react-router-dom
- 地圖 react-leaflet
